# Frontend foundation

- Shared and low-level helpers live in `src/lib`.
- Global state lives in `src/stores` (zustand).
- Shared validation lives in `src/schemas`.
- Mock datasets and handlers live in `src/mocks`.
- HTTP primitives live in `src/services`.

Naming conventions:

- Files and folders: kebab-case
- React component names: PascalCase
- Store hooks: useXStore
- Schemas: *Schema
